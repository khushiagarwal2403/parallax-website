# CSS 3D Parallax Website

A simple parallax scrolling website built using HTML and CSS.

## Features

- 3D Parallax Effect
- CSS Perspective
- CSS Transform
- Fixed Background Sections
- Responsive Layout

## Technologies Used

- HTML5
- CSS3

## Concepts Learned

- Perspective
- Transform Style Preserve 3D
- TranslateZ
- Scale
- Background Attachment Fixed
